# Prime Number Checker

A simple Python program that checks whether a number entered by the user is prime or not.

## Features
- Accepts a number input from the user
- Checks if the number is prime
- Prints whether the number is prime or not

## How to Run
1. Clone the repository
   ```bash
   git clone https://github.com/yuvsharma723/code-playground.git

2. Navigate to the project folder
   cd code-playground

3. Run the Python script
   python prime_checker.py
   **Example Output**
  Enter a number: 7
  7 is prime!
  Enter a number: 8
  8 is not prime!

  Python 3.x
