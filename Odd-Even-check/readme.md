# 🔢 Odd-Even Checker 

A robust Python script designed to determine whether an integer is **Even** or **Odd**. This project is part of my daily coding streak where I focus on mastering Python fundamentals, error handling, and user interaction.

---

## 🚀 Overview
While the logic for checking even or odd numbers is straightforward, this script focuses on **User Experience (UX)** and **Stability**:
* **Error Resilience:** Uses `try-except` blocks to handle invalid inputs (like letters or symbols) without crashing the program.
* **Interactive Loop:** Includes a "Check Again" feature using a `while` loop, allowing for multiple checks in one go.
* **Clean Logic:** Implements the modulo operator (`%`) and Python's ternary operator for efficient result formatting.

## 🛠️ Technical Details
- **Language:** Python 3.x
- **Concepts Used:** - Variable Handling
  - Type Casting (`int()`)
  - Exception Handling (`try...except`)
  - Conditional Statements
  - While Loops
  - String Manipulation (`lower()`)

## 📖 How to Run
1. Clone this repository or download the script.
2. Open your terminal or command prompt.
3. Run the following command:
   ```bash
   python odd_even_checker.py
   ```
